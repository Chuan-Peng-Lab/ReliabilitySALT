The folder includes the following scripts:

## Sim_resampling.Rmd
This script is used to generate the pilot data in stage 1 registration.

## Proposed_analysis_pilot_data.Rmd
This script contains the proposed analysis illustrated in stage 1 registration. We ran the pilot data through our proposed statistical analysis to see whether our proposed analysis code is appropriate for the secondary data structure.

## Test_retest_PilotDData_DDM.py
This script contains the python code to fit the pre-processed pilot data into the drift diffusion model (DDM).
 
## simple_contrast.R
This script is used as a source function in the HLM.
